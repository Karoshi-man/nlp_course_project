# Правила розмітки для витягування навичок (Skill Extraction)

## Сутності для витягування
Ми розмічаємо одну категорію сутностей: **SKILL**.

### SKILL (Технологія / Навичка)
**Включає:**
- Мови програмування (Python, C++, Java)
- Бібліотеки та фреймворки (Pandas, PyTorch, Django, React)
- Бази даних (PostgreSQL, MongoDB)
- Хмарні платформи (AWS, Azure, GCP)
- Інструменти (Docker, Kubernetes, Git)
- Концепції CS/DS (OOP, Machine Learning, NLP, CI/CD)

**Не включає:**
- Soft skills (комунікація, робота в команді, рівень англійської)
- Загальні слова (програмне забезпечення, база даних, комп'ютер, алгоритм)
- Назви компаній (якщо це не назва їхнього інструменту)

### Приклади
1. "Вимагається досвід роботи з **Python** та **Pandas**." -> [SKILL, SKILL]
2. "Знання хмарних сервісів **AWS**." -> [SKILL]
3. "Хороші комунікативні навички та рівень англійської upper-intermediate." -> [NO TAGS]
